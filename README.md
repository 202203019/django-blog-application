# Django Blog Project

A fully functional **Django Blog application** with user authentication, profile management, and CRUD operations for posts. Built with Django, Bootstrap, and Crispy Forms.

## Features

- User registration, login, and logout
- User profiles with profile picture upload
- Create, read, update, and delete (CRUD) blog posts
- Home page with a list of posts
- Post detail pages
- Responsive design using Bootstrap
- Admin panel for managing users and posts
- Image handling for user profiles and posts
- Form validation and secure password management

## Technologies Used

- **Backend:** Python, Django
- **Frontend:** HTML, CSS, Bootstrap
- **Forms:** Django Crispy Forms
- **Database:** SQLite (default, can be switched to PostgreSQL/MySQL)
- **Version Control:** Git & GitHub
